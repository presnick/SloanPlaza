## ðŸ“… Monthly Tasks

### How to Post Meeting Minutes
1.  Go to **Google Drive**.
2.  Open the folder: `Sloan Plaza Documents` > `02 Meeting Minutes`.
3.  **Drag and Drop** the PDF/Word file of the minutes into this folder.
    *   *That's it! The website updates automatically.*

### How to Post Financial Reports
1.  Go to **Google Drive**.
2.  Open the folder: `Sloan Plaza Documents` > `01 Financials` > `Monthly Reports`.
3.  **Drag and Drop** the new report PDF here.

### How to Update the Management Report
*If you are using a static link:*
1.  Upload the new "Management Report" PDF to Drive (Folder `04 Management`).
2.  Right-click the file > **Share** > **Copy Link**.
3.  Go to the **Site Editor**.
4.  Click the "Management Report" text/button.
5.  Click the **Pencil Icon** (Edit Link) and paste the new link.
6.  Click **Publish** (Top Right) to save.

## ðŸ‘¥ Community Updates

### How to Add an Event to the Calendar
1.  Go to **calendar.google.com**.
2.  Make sure you are selecting the **Sloan Plaza Calendar** (not your personal one) when creating the event.
3.  Type the event name (e.g., "Board Meeting").
4.  Set the time.
5.  Click **Save**.
    *   *The website updates automatically.*

### How to Update the Resident Directory
*Since this is a Google Sheet, you don't need to open the website builder.*
1.  Go to **Google Drive**.
2.  Open `Sloan Plaza Documents` > `05 Community` > `Resident Directory (Sheet)`.
3.  Add/Remove rows or edit phone numbers directly in the spreadsheet.
4.  The website will reflect changes automatically.

### How to Add a New Resident (Access)
1.  Go to **Google Groups** (Link above).
2.  Open the **allresidents@sloanplaza.com** group.
3.  Click **Members** > **Add Members**.
4.  Type their email address and click **Add**.
    *   *They now have access to both the Site and the Files (as long as you shared them with this group).*

### How to Add a "Pet of the Month"
1.  Go to the **Site Editor**.
2.  Scroll down to the "Community" section (Pet Carousel).
3.  Click on the photo carousel.
4.  Click the **Gear Icon** (Settings).
5.  Click **Upload Image** (or Select Image) to add the new pet.
6.  Click **Update**.
7.  **IMPORTANT:** Click **Publish** at the top right to make it live.

## ðŸ› ï¸ Occasional Updates

### Updating Board Members
1.  Go to the **Site Editor**.
2.  Scroll to the **Board of Directors** section.
3.  **To Change a Name**: Just click the text and type.
4.  **To Change a Photo**:
    *   Click the old photo.
    *   Click the **Three Dots** > **Replace Image** > **Upload**.
5.  Click **Publish**.

### Changing the "Alert" Header
1.  Go to the **Site Editor**.
2.  Scroll to the top header text works.
3.  Edit the text (e.g., "NEXT MEETING: FEB 20").
4.  Click **Publish**.
