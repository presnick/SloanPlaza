# Sloan Plaza Website: Management Guide

## How to Update Meeting Minutes
1.  **Log in to GitHub.**
2.  Navigate to the generic repository page.
3.  Click on the folder `public` > `minutes`.
4.  **To Upload a New File:**
    *   Click the **"Add file"** button (top right of file list).
    *   Select **"Upload files"**.
    *   Drag your PDF file onto the page.
    *   **Crucial Step:** Scroll down to "Commit changes" and click the green **Commit changes** button.
5.  **Done!** The website will update automatically in about 2-3 minutes.

## How to Check if it Worked
1.  Visit [your website URL] (e.g., https://presnick.github.io/SloanPlaza/).
2.  Log in to the Residents Area.
3.  Check the Meeting Minutes list. Your new file should appear there.
